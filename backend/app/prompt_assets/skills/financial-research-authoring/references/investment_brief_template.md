# 投资简报模板（Investment Brief Template）

基于 Kimi Research 投资简报风格。适用于：重点公司快速覆盖、短期催化剂驱动的交易性报告。

## 报告头部信息栏（单行，紧凑）

```
[公司名]    市值: ¥[X,XXX]亿    行业: [行业]    PE(TTM): [XX.X]x    报告日期: [YYYY-MM-DD]
```

## 大标题（判断句）

```
[公司名]
[大标题：一句话核心判断，10-20字，如"全球新能源龙头，利润承压待修复"]
[副标题：核心财务数据，如"2025年归母净利润326亿元同比-19%，海外扩张成第二增长曲线"]
```

## 核心观点（紧凑段落，120-180字）

必须在这一段包含：营收增速、净利润变化、单项亮点、单项风险、前瞻展望、估值判断。

## 催化剂日历

```
| 日期 | 事件 | 重要性 |
|------|------|--------|
| YYYY-MM-DD | [事件1] | 高/中/低 |
| YYYY-QX | [事件2] | 高/中/低 |
| YYYY-MM | [事件3] | 高/中/低 |
```

重要性定义：
- **高**：直接影响盈利预测或估值（如季报/重大产品发布/监管决定）
- **中**：间接影响情绪或预期（如行业数据/竞对动态）
- **低**：信息型事件（如投资者日/管理层会议）
