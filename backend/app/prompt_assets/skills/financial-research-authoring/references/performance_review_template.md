# 业绩点评报告模板（Performance Review Template）

基于东方资本/卖方研报风格。适用于：上市公司年报、半年报、季报公告后的即时点评报告。

## 报告头部

```
[机构名称]
业绩点评  [YYYY年M月D日]
[一级行业] / [二级行业]

[公司名]（[股票代码].XX）
[年份]年[年报/半年报/季报]业绩点评

[副标题：营收概况] + [利润概况]；
[第二行副标题：核心亮点]
```

## 报告导读（报告开头必须段落，3-5句）

```
[公司名]发布[年份]年[报告类型]，全年实现营业收入[X,XXX]亿元，
同比[增/降][X.X]%；归母净利润[XXX]亿元，同比[增/降][X.X]%；
基本每股收益[X.XX]元。[2-3句说明背景和总体判断，点明业绩韧性或承压原因]。
```

## 投资要点（4-6条 bullet，每条一个核心判断）

```
○ 业绩[符合/超/低于]预期：[YYYY]年归母净利润[XXX]亿元，同比[±X.X]%，[评价]。
○ [核心板块1]：[结论]，[量化数据]。
○ [核心板块2]：[结论]，[量化数据]。
○ [经营现金流/资产负债]：[结论]，[数据支撑]。
○ 盈利预测上/下调：上调/维持/下调[YYYY+1E/YYYY+2E]归母净利润预测至[XXX/XXX]亿元。
```

## 第1节：业绩概览

**节标题格式**：`1. 业绩概览：[10字内结论句，含方向和数字]`

内容必须包含：
1. 核心财务数据一览表（见 SKILL.md 格式）
2. 季度分布分析（上半年/下半年/单四季度）
3. 与预期对比（高于/符合/低于市场预期）

## 第2节：分业务板块分析

每个板块的写作模板：
```
2.X [板块名]：[结论句，含营收增减率和盈利变化]

[板块名]板块[YYYY]年实现营业收入[XXX]亿元，同比[增/降][X.X]%；
实现经营利润[XXX]亿元，同比[增/降][X.X]%。

[量：产量/销量/规模数据，同比变化]
[价：价格/单位成本，同比变化]
[质：毛利率/EBIT margin，同比变化]
[因：解释量/价/质变化的根本原因]
```

## 第3节：盈利能力与费用分析

**必须包含的指标**（含同比变动方向 pct）：
- 综合毛利率：X.X%，同比±X.X pct
- 净利率：X.X%，同比±X.X pct
- 销售费用率：X.XX%，同比±X.XX pct
- 管理费用率：X.XX%，同比±X.XX pct
- 财务费用率：X.XX%，同比±X.XX pct
- 研发费用率：X.XX%，同比±X.XX pct（如适用）

**图表**：`[CHART: combo | title="[公司名]盈利能力指标与费用率变化" | ...]`

## 第4节：现金流与资产负债

**经营现金流**：
- 经营活动现金流净额：[XXX]亿元，同比±X.X%
- 经营现金流/净利润：X.X倍（现金含量评价）

```markdown
| 股东权益（亿元）| | | |
| 资产负债率 | | | |
| 货币资金（亿元）| | | |
```

## 第5节：盈利预测与估值

**三年预测表**（见 SKILL.md 格式）

**估值说明段落**（必须含以下要素）：
- 当前股价
- 使用的估值方法（PE/PB/EV-EBITDA）
- 给予的目标估值倍数及理由
- 目标价（元/股）
- 较当前涨跌幅
- 评级（买入/增持/中性/减持/回避）

## 第6节：风险提示（≥4条）

每条格式：
```
（X）[风险名称]：[风险描述，含量化影响，如"油价每下跌10美元/桶，
影响上游板块盈利约[XX]亿元"。]
```

必须包含的风险类别：
- 宏观/行业系统性风险（如油价、利率、汇率）
- 竞争风险（市场份额、定价压力）
- 政策/监管风险
- 执行风险（项目落地、成本超支）
- 财务/流动性风险（如适用）

## 报告尾部（必须包含）

```markdown
---
**分析师声明**
本报告基于作者专业理解独立出具，结论不受第三方授意或影响。
所有数据来源于公司公告及公开信息，不保证准确性和完整性。

**投资评级说明**
- 买入：未来6个月相对指数涨幅 > +20%
- 增持：未来6个月相对指数涨幅 +10% ~ +20%
- 中性：未来6个月相对指数涨幅 -10% ~ +10%
- 减持：未来6个月相对指数涨幅 -20% ~ -10%
- 回避：未来6个月相对指数涨幅 < -20%

**免责声明**
本报告仅供参考，不构成投资建议。投资者应独立判断，自行承担风险。
过往业绩不代表未来表现。
---
```

## 字数与长度参考

| 节 | 参考字数 |
|---|---------|
| 报告导读 | 150-250字 |
| 投资要点 | 200-300字（4-6条）|
| 第1节业绩概览 | 400-600字 + 1张表 |
| 第2节板块分析 | 每板块200-400字，共600-1200字 |
| 第3节费用分析 | 300-500字 + 1张图 |
| 第4节现金流 | 300-450字 + 1张表 |
| 第5节盈利预测 | 200-400字 + 1张表 |
| 第6节风险提示 | 每条100-150字 × 4-6条 |
| **合计** | **3,000-5,000字** |
