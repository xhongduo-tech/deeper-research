# Financial Research Authoring

## Kimi-Style Contract Adapter

This skill follows the shared Kimi-style execution contract in `../_shared/kimi_style_skill_contract.md`.

**职责：生成机构研报级别的金融分析报告，不写通用商业分析，只做量化驱动的研究报告。**

## 报告类型选择

| 报告类型 | 触发关键词 | 结构模板 | 参考文件 |
|---------|----------|---------|---------|
| 业绩点评 | 年报/季报/业绩点评/earnings review | `references/performance_review_template.md` | 中国石油2025年报风格 |
| 投资简报 | 投资简报/investment brief/投资逻辑 | `references/investment_brief_template.md` | 比亚迪BYD风格 |
| 深度报告 | 深度/初次覆盖/首次报告 | `references/deep_report_template.md` | 完整8-12节结构 |

## 业绩点评结构（Performance Review）

```
[报告头部]
  机构名称 | 报告日期 | 行业分类 | 股票代码
  分析师姓名 | 投资评级：增持/中性/减持/回避

[报告导读]（3-4句核心摘要，含关键数字）

[投资要点]（4-6条带数字的 bullet）
  ○ 业绩符合/超/低于预期 + 数字
  ○ 核心驱动因素 + 数字
  ○ ...

1. 业绩概览：[标题必须是结论句，含数字]
   - 核心财务数据一览表（营收/毛利/净利润/EPS + 同比）
   - 季度趋势分析

2. 分业务板块分析
   2.1 [板块名]：[结论标题]
   2.2 [板块名]：[结论标题]
   ...

3. 盈利能力与费用分析
   - 毛利率/净利率/三项费用率 + 同比变动
   - 图表：盈利指标趋势图

4. 现金流与资产负债
   - 经营/投资/筹资现金流
   - 关键资产负债指标表

5. 盈利预测与估值
   - 三年预测表（YYYY A, YYYY+1 E, YYYY+2 E）
   - 当前估值 PE/PB/PS 与历史比较
   - 投资评级及目标价逻辑

6. 风险提示（≥4条，每条有量化影响说明）

[分析师声明 + 投资评级定义 + 免责声明]
```

## 投资简报结构（Investment Brief）

```
[报告头部信息栏]
  公司名 | 市值 | 行业 | PE(TTM) | 报告日期

[标题行]
  大标题（一句判断，如"全球新能源龙头，利润承压待修复"）
  副标题（核心数据）

[核心观点]（单段，120-180字，含所有关键数字）

[左侧主体]
  投资逻辑（短/中/长期分拆）
  公司概览（成立/总部/主营/员工/实控人）
  业务结构表（板块/营收/占比/毛利率）
  产业链分析（上/下游）
  财务分析（3年历史数据表）
  偿债能力分析

[右侧边栏]
  核心交易数据（价格/52周高低/超额收益/市值/PE/PB/PS/换手率）
  股价走势图（52周，vs 指数归一化）
  估值分析（当前 vs 5年均值 + 判断）
  催化剂日历（日期/事件/重要性）
  可比公司估值对比表

[情景分析]
  3情景表（乐观/基准/悲观 + 概率/净利润预测/目标价/核心假设）
  概率加权目标价计算公式

[风险提示]（4-6条）
[免责声明]
```

## 投资评级定义（必须写入报告）

| 评级 | 定义 |
|------|------|
| **买入** | 未来6个月相对指数涨幅 > +20% |
| **增持** | 未来6个月相对指数涨幅 +10% 至 +20% |
| **中性** | 未来6个月相对指数涨幅 -10% 至 +10% |
| **减持** | 未来6个月相对指数涨幅 -20% 至 -10% |
| **回避** | 未来6个月相对指数涨幅 < -20% |

## 量化语言规范

### 必须量化的表达

| 避免 | 要求 |
|------|------|
| "盈利能力较强" | "净利润率 6.0%，高于行业均值 4.1% 约 2个百分点" |
| "增长较快" | "同比增长 4.8%，高于行业平均增速 2.3pct" |
| "估值合理" | "当前 PE 23.3x 处于5年历史约40%分位" |
| "风险较大" | "若碳酸锂继续上涨 30%，单车成本额外增加约 2,000-4,000 元" |

### 单位规范
- 大金额：**亿元**（不用万元、百万）
- 小金额/单价：**元**（如EPS、单车利润）
- 占比/变动：**百分点（pct）** 而非"%"（用于同比描述）
- 利润率：**X.X%** 格式（保留一位小数）
- 市盈率：**XX.Xx**（保留一位小数，加"x"后缀）
- 目标价：**元/股**（明确说明基准时间）

## 估值方法论（必须说明）

当给出目标价时，必须明确说明估值方法：

```
估值方法：给予[公司名] [YYYY+1E] PE [XX]x，对应目标价 [XX] 元/股。
估值逻辑：[公司名]属于[行业]，历史平均 PE [XX]x，考虑到[具体因素]，
给予[折溢价说明]，预测 YYYY+1E EPS [XX] 元，得出目标价 [XX] 元。
当前股价 [XX] 元，上行空间 [X%]，维持[评级]评级。
```

若使用 PB 法、DCF 法、SOTP法，同样需要说明具体参数和计算过程。

## 禁止行为

- **禁止编造任何财务数字**（无来源的数字一律用 `[PLACEHOLDER: ...]`）
- **禁止给出无计算依据的目标价**（必须说明估值方法和参数）
- **禁止省略风险提示**（必须≥4条，每条量化影响）
- **禁止非量化的买卖建议**（"值得关注"等模糊表达不符合合规要求）
- **禁止省略免责声明和分析师声明**
- **禁止混用货币单位**（全文统一）
- **禁止在结论中引入新数据**（结论只是对正文的总结）
