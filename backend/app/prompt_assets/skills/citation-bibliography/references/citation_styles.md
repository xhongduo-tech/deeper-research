# Citation Style Rules

## GB/T 7714

Use for Chinese academic and policy-adjacent documents when unspecified.
Preserve author, title, source, year, URL/DOI when available.

## APA

Use author-year in text and full references at the end.

## IEEE

Use numeric brackets in order of appearance.

## Business Reports

Prefer source notes under figures/tables and a compact source list in appendix.

## Missing Metadata

If metadata is incomplete, cite the uploaded file and section/page/table anchor.
Do not invent missing fields.
