# Claim Source Map

```json
{
  "claim": "",
  "claim_type": "number|trend|causal|definition|quote|recommendation",
  "source_id": "",
  "source_anchor": "",
  "support_level": "direct|calculated|inferred|assumption",
  "note": ""
}
```

## Support Levels

- `direct`: source states the claim.
- `calculated`: derived from source data.
- `inferred`: reasoned from multiple pieces of evidence.
- `assumption`: not sourced; must be labeled.
