# Content Isolation Rules

Reference examples often contain proprietary or irrelevant facts. The miner must
block them from downstream generation unless the user marks them as source
evidence for the current task.

## Blocked By Default

- Names, companies, departments, products, clients, projects.
- Dates, financial numbers, KPIs, rankings, market size, case results.
- Conclusions, recommendations, claims, and legal obligations.
- Sensitive phrases, slogans, internal policy text, and confidential wording.

## Allowed By Default

- Section roles.
- Heading depth and numbering pattern.
- Table/figure types.
- Caption style.
- Paragraph density and tone.
- Appendix and reference placement.

## Handoff Format

Return blocked facts as short labels, not verbatim text.
