# Fact And Logic Checks

## User Need Adherence

Fail the artifact with `P0` if it mostly summarizes uploaded material, follows
template demo content, or uses a generic report structure while missing the
specific user request.

Check:

- The title reflects the input-box topic, not merely the first uploaded file.
- Each major section, slide, or sheet maps to a user requirement or necessary support.
- Explicitly requested entities, metrics, regions, periods, audiences, and angles are covered.
- Irrelevant uploaded-file sections are excluded unless the user asked for a full summary.
- Missing source material needed to answer the user is disclosed as a gap, not filled with generic claims.

## Fact Checks

- Every headline claim has evidence, calculation, or assumption status.
- Numeric claims include unit, period, denominator, and source where relevant.
- Rankings, growth rates, shares, and forecasts have a clear basis.
- Current or "latest" statements use current sources or avoid current wording.
- Source labels match the actual file, page, section, table, or sheet.
- Template demo content does not appear as final evidence.

## Calculation Checks

- Formulas use the correct rows, columns, and denominators.
- Percentages and percentage-point changes are not confused.
- Currency, scale, and unit conversions are visible.
- Totals reconcile with source data or are explained.
- Forecasts separate assumptions from historical facts.

## Logic Checks

- Recommendations follow from findings.
- Findings do not contradict each other.
- The document or deck answers the user's original question.
- Risks and caveats do not invalidate the main conclusion without being addressed.
- Each section, slide, or sheet has a clear purpose.

## Common Failure Patterns

- "显著/大幅/领先" without baseline.
- A causal recommendation from descriptive data only.
- A conclusion that introduces new evidence not shown earlier.
- Inconsistent names for the same entity or metric.
- Executive summary claims absent from the body.
