# Format And Export Checks

## PPT Checks

- Slide count matches the requested range or justified plan.
- Titles, body text, charts, tables, and source notes do not overlap or clip.
- Slide titles form a coherent story when read alone.
- Preview and exported PPTX use the same SlideSpec and slide order.
- Text, tables, charts, and shapes remain editable where possible.
- Speaker notes are present when promised and do not hide essential content.

## Word Checks

- Heading levels, numbering, table of contents, headers, and footers are consistent.
- Tables fit page width or move to appendix.
- Captions stay near figures and tables.
- No orphan headings or broken section flow.
- Citations/source notes are formatted consistently.
- DOCX opens without missing images, broken links, or collapsed layout.

## Excel Checks

- Sheet names are clear and stable.
- Inputs, calculations, outputs, and charts are separated.
- Formulas reference intended ranges and do not contain hard-coded accidental constants.
- Filters, number formats, frozen panes, and table styles support review.
- Exports preserve formulas, chart readability, and source notes.

## General Checks

- File names and visible titles match the user topic.
- Required output format is produced.
- Language, tone, and terminology are consistent.
- No placeholder text, TODOs, or template filler remains.
