# Skill Depth Rubric

## Shallow Skill

- Has a generic description.
- Says "write well" without stage outputs.
- No inputs, outputs, or QA rules.

## Good Skill

- Specifies domain and use cases.
- Has workflow and checklist.
- Names dependent skills.

## Excellent Skill

- Includes schemas, quality gates, failure modes, handoff rules, and test prompts.
- Separates evidence, style, and template constraints.
- Prevents leakage from reference examples.
- Can be inspected in the UI and routed by Chief.
