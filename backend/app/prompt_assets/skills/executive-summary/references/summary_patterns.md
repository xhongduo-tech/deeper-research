# Executive Summary Patterns

## Research Report

1. Context.
2. Core findings.
3. Evidence snapshot.
4. Implications.
5. Recommendations.

## Business Plan

1. Opportunity.
2. Solution.
3. Traction or evidence.
4. Business model.
5. Funding/resource ask.

## Feasibility Study

1. Project conclusion.
2. Technical feasibility.
3. Economic feasibility.
4. Organizational/legal feasibility.
5. Risk controls.

## Technical Proposal

1. Current problem.
2. Target architecture.
3. Implementation path.
4. Risks/dependencies.
5. Acceptance criteria.
