# Slide Storyline Contract

## Default Executive Arc

1. Cover: exact topic, owner/context, date if useful.
2. Executive takeaway: the main answer in one slide.
3. Situation or baseline: why this topic matters now.
4. Evidence block: 3-6 slides grounded in files, data, or research.
5. Implication: what the evidence means for the audience.
6. Options or recommendation: choices, trade-offs, or proposed direction.
7. Action plan: owners, timeline, milestones, and risks.
8. Appendix: methods, source table, dense data, or backup details.

## Alternative Arcs

- **Problem-solving**: problem -> root cause -> options -> recommendation -> implementation.
- **Research briefing**: question -> method -> findings -> implications -> watchlist.
- **Progress report**: objectives -> current status -> achievements -> issues -> next period plan.
- **Roadshow**: market pain -> solution -> traction -> business model -> ask.

## Page Budget

- 5-7 slides: compress to takeaway, evidence, implication, action.
- 8-12 slides: standard executive arc.
- 13+ slides: add appendix and split evidence by theme.

## Rules

- Every slide answers a decision question.
- Use one idea per slide; split when two audiences or decisions compete.
- Avoid generic labels like "背景介绍" when a conclusion title is possible.
- Put methods, caveats, and long explanations in notes or appendix.
