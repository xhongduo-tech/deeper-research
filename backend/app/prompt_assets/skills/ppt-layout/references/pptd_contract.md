# PPTD Contract

`.pptd` files define visual behavior only.

## May Define

- Palette and color roles.
- Font family, size ranges, and hierarchy.
- Slide density preferences.
- Preferred layout families.
- Spacing, margins, and grid behavior.
- Decorative motifs or illustration style.
- Export constraints.

## Must Not Define

- Final facts, claims, titles, numbers, recommendations, or source evidence.
- User-specific industry examples unless the user explicitly requested template content.
- Content that overrides uploaded files or user instructions.

## Template Handling

- Extract style rules and ignore demo copy.
- Keep brand feel while adapting layout to actual slide roles.
- If a template lacks a needed layout, compose within its grid and palette.
- Record when content had to be split because the template density was too low.
