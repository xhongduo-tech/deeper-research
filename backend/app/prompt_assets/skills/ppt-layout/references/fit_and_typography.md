# Fit And Typography

## Fit Rules

- Split a slide when content exceeds six bullets or one dense table.
- Convert paragraphs into bullets, callouts, tables, or speaker notes.
- Shorten labels before shrinking text below readability thresholds.
- Keep title, body, caption, and source labels from overlapping.
- For CJK text, test longer words and mixed English identifiers.

## Typography Ranges

- Title: 28-36pt.
- Subtitle: 16-22pt.
- Body: 14-18pt.
- Table body: 10-14pt, only if readable.
- Metric: 44-72pt.
- Captions and sources: 9-11pt.

## Visual Hierarchy

- One dominant element per slide.
- Use contrast for meaning, not decoration.
- Keep recurring elements in stable positions across the deck.
- Do not let charts, tables, or images crowd the title.

## Export Checks

- Preview and PPTX have the same page count and slide order.
- Text remains editable when exported.
- Fonts, images, charts, and tables render without clipping.
