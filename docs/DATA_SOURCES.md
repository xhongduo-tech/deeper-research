## 四、需要你解决的API清单（按优先级排序）

### 🔴 高优先级（影响金融核心数据）

1. **Tushare Pro API Token**（A股/港股/基金/宏观数据）
   - 官网: https://tushare.pro
   - 注册免费，积分制，日常用量足够
   - 用途: A股K线、财务数据、基金净值、宏观经济

2. **Alpha Vantage API Key**（美股数据）
   - 官网: https://www.alphavantage.co
   - 免费版: 5次/分钟，500次/天
   - 用途: 美股行情、技术指标

3. **国家统计局 EasyQuery API**（已有connector，确认可用）
   - 官网: https://data.stats.gov.cn/easyquery.htm
   - 免费
   - 用途: GDP/CPI/PMI等宏观数据

### 🟡 中优先级（影响学术/企业深度）

4. **CNKI API / 机构账号**（中文学术论文）
   - 需高校/研究机构授权
   - 用途: 中文核心期刊论文

5. **企查查API / 天眼查API**（企业工商/司法数据）
   - 付费，约几千元/年
   - 用途: 企业注册信息、司法风险

### 🟢 低优先级（可替代/延后）

6. **Wind/iFinD API**（券商研报）
   - 非常贵（数万元/年）
   - 替代方案: 网页爬取公开研报摘要
